# Code of Conduct

## Our standard

We are committed to a welcoming, respectful, and harassment-free community. Participate professionally, assume good faith, welcome correction, and focus criticism on ideas and behavior rather than people.

Unacceptable behavior includes harassment, discrimination, threats, intimidation, sexualized conduct, doxxing, malicious disruption, and publishing private information without consent.

## Enforcement

Report concerns privately to the maintainers. If repository-host private reporting is enabled, use it; this document does not claim that a private channel is currently configured. Reports will be reviewed confidentially where possible. Maintainers may clarify, warn, remove content, temporarily restrict participation, or permanently ban behavior that violates this code.

This code applies in project spaces and when someone represents the project.
