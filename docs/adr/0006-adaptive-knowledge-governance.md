# ADR 0006: Adaptive knowledge governance

**Status:** Accepted

## Decision

Use bounded curated memory with FTS and a full retrospective on every task. Learning follows propose, evaluate, approve, activate, and rollback. Knowledge is scoped and deletion-aware; routing proposals are shadow-evaluated before activation.

## Consequences

Knowledge can improve assistance without silently changing policy. Memory growth and learned behavior remain bounded, reviewable, and reversible. Code intelligence and context optimization are separate decisions.
